第1章

GPUが見えるか確かめたい

最初の一歩は、「この環境で GPU が使えるのか」を自分の目で確かめることです。

今日は何をしたい？

ROCm 版 PyTorch が入っていて、いまの PC から GPU が見えているかを短いコードで確認します。まずここが通ると、この先の章で「CPU だけで動いていた」という遠回りを減らせます。

最小コード

import torch

print("HIP:", torch.version.hip)
print("GPU available:", torch.cuda.is_available())

if torch.cuda.is_available():
    print("Device:", torch.cuda.get_device_name(0))

torch.version.hip が見えて、torch.cuda.is_available() が True なら、まずは出発できます。

どこがROCm？

ROCm 版 PyTorch でも、歴史的な理由で torch.cuda という名前の API を使う場所があります。名前はそう見えても、ROCm 環境では下で HIP/ROCm 側の処理につながります。

ここで出てきたPython

  • import でライブラリを読み込む
  • print() で確認したい値を出す
  • if で「GPU が見えているときだけ」を分ける

よくあるつまずき

  • GPU available: False なら、PyTorch の build か ROCm の見え方を先に確認します。
  • torch.version.hip が None なら、ROCm 向けではない PyTorch が入っていることがあります。
  • GPU 名が出なくても慌てず、まずは False なのかエラーなのかを分けて見るのが大事です。

1分演習

print("GPU available:", ...) の行だけを残して実行し、まずは True か False かを自分の環境で確かめてみましょう。
