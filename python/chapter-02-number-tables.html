第2章

数字を表として持ちたい

ここでは NumPy を使って、数字の並びを「表」として見る感覚を作ります。

今日は何をしたい？

AI や GPU の話では、数字は 1 個ずつよりも「まとまった表」で出てくることが多いです。まずは NumPy 配列を作って、shape を「表の大きさ」として読めるようにします。

  • この章のゴールは、配列を作る → shape を読む の2点です。
  • Python 文法が未修でも大丈夫なように、必要な記号だけこの章で説明します。
  • ROCm や GPU の本番利用は次章以降。ここでは「数字を表で扱う感覚」を作ります。

実行前チェック（20秒）

  • 第1章で使った Python をそのまま使います（例: ./.venv/bin/python）。
  • numpy が入っているか不安なら、pip show numpy を実行します。

確認コード（文法未修でも読める版）

import numpy as np

x = np.array([[1, 2, 3],
              [4, 5, 6]], dtype=np.float32)

print("x:")
print(x)

print("shape:", x.shape)
print("dtype:", x.dtype)
print("rows:", x.shape[0])
print("cols:", x.shape[1])

この例では、2 行 3 列の表を 1 つのまとまりとして持っています。(2, 3) と見えたら、「2 行 3 列」と読めば大丈夫です。

実行コマンド（コピペ用）

./.venv/bin/python -c 'import numpy as np; x=np.array([[1,2,3],[4,5,6]],dtype=np.float32); print("x:"); print(x); print("shape:", x.shape); print("dtype:", x.dtype); print("rows:", x.shape[0]); print("cols:", x.shape[1])'

まずはこの1行で、出力を見て「shape の読み方」を確認します。

実行すると何が出る？

x:
[[1. 2. 3.]
 [4. 5. 6.]]
shape: (2, 3)
dtype: float32
rows: 2
cols: 3
  • shape: (2, 3) は「2 行 3 列」です。
  • dtype: float32 は「小数を32ビットで持つ」型です。
  • rows と cols を別で出すと、行列の向きで混乱しにくくなります。

どこがROCm？

この章のコードは CPU 側ですが、ここで作る「shape を読む習慣」がそのまま PyTorch と GPU の章につながります。ROCm でも、まずは何行何列なのかを読めることが大事です。

第1章で確認した GPU は、次の章以降で「この形のデータをどう計算するか」に使います。つまり第2章は、GPU計算の前に必要な土台です。

ここで出てきたPython（未修向けミニ文法）

  • import numpy as np は「NumPy を np という短い名前で使う」宣言です。
  • = は「代入」。右側の結果を左側の変数に入れます。
  • 入れ子の [] で表の形を作ります（1行が1つの []）。
  • x.shape や x.dtype は、x の情報を読む書き方です。
  • x.shape[0] は1つ目（行数）、x.shape[1] は2つ目（列数）です。

結果パターン別の次の一手

  • パターンA: shape: (2, 3) と出た → この章はクリア。次章へ進んでOK。
  • パターンB: ModuleNotFoundError: No module named numpy → numpy未導入。pip install numpy 後に再実行。
  • パターンC: shape が想定と違う → 入れ子の [] の行数と列数を見直します。
  • パターンD: 実行はできたが意味が曖昧 → rows と cols を必ず表示して確認します。

よくあるつまずき

  • 各行の長さがそろっていないと、配列化で失敗したり意図しない型になったりします。
  • 例: [[1,2,3],[4,5]] のような不ぞろい形は避けます。
  • (2, 3) を「2 列 3 行」と逆に読まないように気をつけます。
  • dtype は「数字の種類」です。最初は float32 と見えたら十分です。

1分演習

次の順で試します。1) まず shape を予想する。2) 実行して答え合わせする。3) rows/cols でも確認する。

y = np.array([[10, 20],
              [30, 40],
              [50, 60]], dtype=np.float32)
print("shape:", y.shape)

正解は (3, 2) です。ここを迷わず読めれば、第2章の到達目標は達成です。
