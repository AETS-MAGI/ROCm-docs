Python and ROCm 教室

Python and ROCm Course

やりたいことから Python と ROCm を学ぶコース

A course that teaches Python and ROCm through concrete goals

このコースは、変数や関数から順番に入るのではなく、「GPU が見えるか確かめたい」「行列を計算したい」「小さなモデルを動かしたい」という目標から始まります。必要な Python は、その場で少しずつ覚えていけば大丈夫です。

This course does not begin with abstract grammar topics. It starts from concrete goals such as checking the GPU, doing matrix math, and running a small model, while introducing only the Python needed at each step.

このコースが向いている人: Python を全部覚えてから GPU に進むのではなく、まず手を動かしながら NumPy・PyTorch・ROCm のつながりを知りたい人。

This course is for: learners who do not want to master all of Python first, but want to understand how NumPy, PyTorch, and ROCm connect while doing small practical tasks.

まず何を読む？

Where Should You Start?

最初の数章だけでも、かなり多くのページが読みやすくなります。

Even the first few chapters make many other pages easier to read.

  1. GPU が見えるか確かめたい で、ROCm 環境と PyTorch の最初の見え方を確認する
  2. Check whether the GPU is visible to confirm the first PyTorch / ROCm setup signs
  3. 数字を表として持ちたい と 表どうしを計算したい で、shape と行列の読み方に慣れる
  4. Hold numbers as tables and Multiply tables to get comfortable with shapes and matrices
  5. NumPy から PyTorch へ移りたい で、tensor の見方に進む
  6. Move from NumPy to PyTorch to start reading tensors naturally

このコースで身につくこと

What This Course Builds

コードの読み方

Reading Code

`torch.tensor(...)`、`.to(device)`、`model(x)`、`loss.backward()` のような行を見たときに、「これは何をしている行か」を一つずつ結びつけられるようになります。

You learn to look at lines like `torch.tensor(...)`, `.to(device)`, `model(x)`, and `loss.backward()` and understand what each one is doing.

shape の感覚

Shape Intuition

NumPy の配列、PyTorch の tensor、行列積、Attention の形までを、「数字の表」として追いやすくなります。

You build a clear “tables of numbers” intuition that carries from NumPy arrays to PyTorch tensors, matrix multiplication, and attention shapes.

ROCm とのつながり

The ROCm Connection

Python のコードを書いているときでも、その下で GPU が使われ、rocBLAS や MIOpen のような計算ライブラリにつながっていく感覚をつかめます。

You begin to see how simple Python code can connect downward to GPU work through ROCm libraries such as rocBLAS and MIOpen.

実験コードへの自信

Confidence with Small Experiments

短いスクリプトを読んだときに、「わからない単語だらけで止まる」状態から抜けて、自分で追える範囲が増えていきます。

Instead of stopping at every unfamiliar line, you gradually become able to follow short experiment scripts by yourself.

次に進むなら: 13 章の全体像は 学習ロードマップ で見られます。

Next step: you can see the full 13-chapter path on the course roadmap.

おすすめの前提: GPU そのものや ROCm の全体像がまだぼんやりしているなら、先に 理論ページ を軽く読んでおくと、このコースがかなり追いやすくなります。

Recommended prep: if GPUs and ROCm still feel vague, a quick look at the theory pages makes this course much easier to follow.

少しずつ読み進めれば大丈夫です。全部を一度に理解する必要はありません。 You do not need to understand everything at once. Small steps are enough.
