Python 説明ページ

Python Explanation Pages

このディレクトリは Python スクリプト置き場ではなく、Python の文法や考え方を説明する公開ページを置くための場所です。

This directory is not for Python scripts. It is meant for public pages that explain Python syntax and Python thinking.

この場所の役割: `import`、変数、関数、クラス、配列、shape の読み方など、ROCm や PyTorch を読む前に必要な Python の説明ページをここに置きます。

Role of this place: This is where explanation pages about `import`, variables, functions, classes, arrays, and reading shapes should live before a reader moves on to ROCm or PyTorch.

置かないもの: 実行用スクリプト、運用メモ、雑多な作業メモはここに置きません。そうしたものは `scripts/` か `__tmp/` 側で扱います。

What should not go here: Runnable scripts, operational notes, and rough working memos do not belong here. Those should go under `scripts/` or `__tmp/` instead.
