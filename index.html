GPU・NumPy・PyTorch・ROCm を、やさしくつなげて学ぶサイト

A beginner-friendly site that connects GPU basics, NumPy, PyTorch, and ROCm

このサイトでは、GPU の基本から始めて、NumPy や PyTorch を通して ROCm に近づいていきます。難しい言葉を先に暗記するよりも、「何ができるのか」「どう読むのか」を小さなページで少しずつつかめる構成にしています。

This site starts from GPU basics and gradually leads you toward ROCm through NumPy and PyTorch. Instead of memorising terminology first, it focuses on what you can do and how to read the code step by step.

はじめてなら: まずは 理論ページ で GPU と ROCm の全体像をつかみ、そのあと Python and ROCm 教室 に進むのがおすすめです。

If you're new: start with the theory pages to get the big picture, then move on to the Python and ROCm course.

おすすめの進み方

Suggested Reading Order

迷ったら、次の順で読むとつながりやすいです。

If you're not sure where to begin, this order works well.

  1. GPU のしくみ で、CPU と GPU の違いをつかむ
  2. GPU Basics to understand the difference between CPU and GPU
  3. ROCm の全体構造 で、アプリから GPU までの流れを見る
  4. ROCm Overview to see the path from the app down to the GPU
  5. NumPy特急便 と PyTorch特急便 で、数字の表と tensor の読み方に慣れる
  6. NumPy Express and PyTorch Express to get comfortable with number tables and tensors
  7. Python and ROCm 教室 で、やりたいことベースの 13 章をたどる
  8. Python and ROCm Course to follow the 13-chapter goal-based learning path
