ディレクトリの役割

Directory Roles

迷いやすいディレクトリだけ、先に意味を固定しておきます。

These are the directories that benefit most from explicit role definitions.

  • `theory/` : ROCm / GPU / AI の説明ページ本体。
  • `theory/` : Main explanation pages for ROCm / GPU / AI.
  • `python/` : Python の説明を書くための場所。実行用スクリプト置き場ではありません。
  • `python/` : A place for Python explanation pages. It is not a storage area for runnable scripts.
  • `scripts/` : サイト生成や補助処理のためのスクリプト。
  • `scripts/` : Helper scripts for site generation and supporting tasks.
  • `__tmp/` : 運用メモ、一時メモ、下書き、整理前のメモを置く場所。公開コンテンツの本体はここに置かない方針です。
  • `__tmp/` : Temporary operational notes, drafts, and unpolished working memos. This is not intended to hold the main public content.

Where to store working notes: Operational notes and in-progress memos should go under /home/limonene/ROCm-project/vega-hbmx-investigations/ROCm-docs/__tmp. Keep `python/` for explanation pages only.
